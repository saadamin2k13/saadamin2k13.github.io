Research Projects

🤖

Perspective-taking LLMs for Inclusive Decision Making

Leveraging Large Language Models to enhance social impact by improving decision-making processes, considering user information from various perspectives for more inclusive outcomes.

📊

Bidirectional Evaluation for DRS Systems

Introduced Pars-Gen and Gen-Pars evaluation methodologies for comprehensive DRS-based system assessment, addressing limitations in traditional metrics for structural and linguistic fidelity.

🔄

Task Reversibility in DRS Processing

Explored error propagation in reversible semantic parsing and text generation with DRS, developing innovative pipeline approaches for error correction from multilingual perspectives.

🌐

Multi-Faceted Data Augmentation

Developed novel approaches for augmenting formal meaning representation through Named Entity, Lexical, and Grammatical augmentation strategies for English, Italian, and Urdu.

🇵🇰

Urdu Semantic Processing

Created the first Semantic Resource for Urdu (Urdu Meaning Bank - UMB) and implemented neural semantic parsing and generation tasks with data augmentation for this low-resource language.

🤲

Sign Language Recognition

Developed smart sensor-based data glove prototype for capturing alphabetical and numerical sign postures, generating datasets and utilizing neural networks for static sign gesture classification.
